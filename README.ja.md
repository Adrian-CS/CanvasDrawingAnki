# Kanji Drawing Canvas

> [English](README.md) · [Español](README.es.md) でもご覧いただけます

[Anki](https://apps.ankiweb.net/) 用アドオンです。任意のノートタイプに手書き用の
描画キャンバスを追加し、漢字・ハングルなどの文字をフラッシュカードの復習中に
直接書いて練習できます。デスクトップと**モバイル**の両方で動作します。

---

## 機能

- **独立したキャンバスブロック**をカード内容の下に追加します。既存のテンプレートの
  レイアウトを崩しません。
- **練習用グリッド**: 田字格（4分割）、米字格（4分割＋対角線）、またはグリッドなし。
- 一画ずつ**元に戻す**ボタンと**クリア**ボタン。
- **画数カウンター** — 漢字の画数確認に便利です。
- **モバイル対応**（AnkiDroid / AnkiMobile）。HTML5 Canvas + Pointer Events を
  使用しているため、モバイル側にアドオンは不要です。
- **UI言語の自動検出**: 英語・スペイン語・日本語から、デバイスのロケールに合わせて
  自動的に選択されます。
- 非破壊的: キャンバスはいつでも同じダイアログからテンプレートに対して削除できます。

## インストール

### AnkiWeb から *（推奨）*

1. Anki で **ツール → アドオン → アドオンを取得** を開きます。
2. アドオンコードを入力します *(AnkiWeb 審査後に公開予定)*。
3. Anki を再起動します。

### 手動インストール

1. このリポジトリをダウンロードまたはクローンします。
2. `kanjiDrawingAnki` フォルダを Anki のアドオンフォルダにコピーします
   （`ツール → アドオン → アドオンフォルダを開く`）。
3. Anki を再起動します。

---

## 使い方

1. **ツール → 描画キャンバス…** を開きます。
2. ドロップダウンから変更したい**ノートタイプ**を選択します。
3. リストから**テンプレート**（通常は *Card 1*）を選択し、**キャンバスを追加**を
   クリックします。
4. 通常通り学習します — カードの表面の下部にキャンバスが表示されます。

**推奨する学習フロー:**

```
表面: 意味・読みを確認  →  文字を書く  →  カードをめくる  →  答えと比較
```

**キャンバスを削除する**には: ダイアログを開き、同じテンプレートを選択して
**キャンバスを削除**をクリックします。

---

## 設定

**ツール → アドオン** で *Kanji Drawing Canvas* を選択し、**設定**をクリックします。

| キー | デフォルト | 説明 |
|------|-----------|------|
| `canvas_size` | `300` | キャンバスの一辺の長さ（ピクセル） |
| `grid_type` | `"tian"` | `"tian"`（田）、`"mi"`（米）、または `"none"` |
| `stroke_width` | `3` | ブラシの太さ（ピクセル） |
| `stroke_color` | `"#1a1a1a"` | 筆跡の色（CSS カラー値） |
| `grid_color` | `"#cccccc"` | ガイド線の色 |
| `background_color` | `"#ffffff"` | キャンバスの背景色 |

設定を変更した後は、対象テンプレートでキャンバスを**削除してから再追加**することで
新しい設定が反映されます。

---

## 仕組み

アドオンは、選択したノートタイプの*表面*テンプレートの末尾に、特殊なマーカーコメント
で囲まれた小さな自己完結型の `<div>` + `<script>` ブロックを追記します。
新しいフィールドは作成されず、カードデータも変更されません。キャンバスの状態は
一時的なもので、新しいカードごと・カードをめくるたびにリセットされます。
まるで物理的な練習用紙のように使えます。
